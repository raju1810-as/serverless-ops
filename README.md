# Serverless DevOps Monitoring and Alerting System using AWS

## Overview
This project provides a real-time serverless monitoring dashboard using AWS cloud services.

## Features
- AWS Lambda Monitoring
- CloudWatch Metrics
- SNS Alerts
- Real-time Dashboard
- Error Tracking
- DevOps Observability

## Technologies Used
- React.js
- Node.js
- AWS Lambda
- Amazon CloudWatch
- SNS
- API Gateway

## Run Project

```bash
npm install
node server.js
