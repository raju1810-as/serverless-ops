How to use ServerlessOps

Quick Start
1. Make sure your backend is running: node server.js
2. Click Connect AWS and enter your IAM credentials
3. Generate traffic: node load-test.js
4. Watch metrics update every 60 seconds automatically
Required IAM Permissions
• lambda:ListFunctions, lambda:InvokeFunction
• cloudwatch:GetMetricData, cloudwatch:ListMetrics
• logs:FilterLogEvents, logs:DescribeLogGroups
• dynamodb:ListTables, dynamodb:DescribeTable
• sqs:ListQueues, sqs:GetQueueAttributes
• s3:ListAllMyBuckets
